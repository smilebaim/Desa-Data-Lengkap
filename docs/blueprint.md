# **App Name**: Desa data Connest

## Core Features:

- Indonesia Map Interface: Display a map of Indonesia using Leaflet on the main page ('/') to visualize village locations.
- Village Data Input Module: Allow admin/operator/village officials to input, edit, and delete village data.
- Role-Based Access Control: Implement user roles: Superadmin/developer, admin, operator, village staff. Each role has specific permissions.
- Pelaporan System: The System allows pelaporan activities, ranging from activity planning to post-activity reporting with multimedia documentation.
- Subdomain Management: Allow superadmins/developers to create and manage subdomains for registered villages.
- AI-Powered Content Suggestion Tool: Provide AI-driven suggestions for village-related content based on trends or news, using a reasoning tool to determine the best recommendations.

## Style Guidelines:

- Primary color: Forest green (#388E3C) to represent growth and sustainability.
- Background color: Light beige (#F5F5DC), providing a clean, neutral backdrop.
- Accent color: Burnt orange (#D35400) for call-to-action buttons, creating visual interest.
- Body and headline font: 'PT Sans', a humanist sans-serif providing a modern and accessible feel.
- Use simple, flat icons to represent different village resources and data categories.
- Maintain a clean and intuitive layout with a focus on map and village data visualization.
- Use subtle transitions and animations to enhance user experience when interacting with the map and data.
